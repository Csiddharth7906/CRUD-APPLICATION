# Birthday Wish

A modern, animated, premium birthday wish website built with React, GSAP, Tailwind CSS, and Framer Motion.

## Quick start

```
npm install
npm run dev
```

## Build

```
npm run build
npm run preview
```

## Notes
- Smooth scrolling powered by Lenis.
- Animations by GSAP ScrollTrigger and Framer Motion.
- Tailwind v4 via @tailwindcss/vite plugin.
